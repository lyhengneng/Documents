### 1.4 System of Equations
```
A + B = 76
A - B = 38
```

**Solution:**
1. Add both equations: 2A = 114
2. A = 114 / 2
3. **A = 19**

4. Substitute back: A + B = 76
5. B = 76 - 19
6. **B = 57**

**Final Answer:** A = 19, B = 57
